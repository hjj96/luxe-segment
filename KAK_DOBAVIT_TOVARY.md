# Как добавлять товары в каталог (через таблицу)

Товары можно редактировать как в **Excel-файле** `data/products.xlsx` (рекомендуется), так и в `data/products.csv` (резервный вариант).

---

## Шаг 1: Откройте таблицу

- **Рекомендуемый файл:** `data/products.xlsx`  
- Откройте его в Excel (двойной клик) или загрузите в Google Таблицы (Файл → Импорт).

Если вы редактируете `products.csv`, там разделитель **точка с запятой (;)**.

Если нужно подготовить файл с выпадающими списками (категория, наличие, под заказ), выполните:

```bash
npm run products:xlsx
```

---

## Шаг 2: Колонки таблицы

| Колонка      | Пример                    | Что писать |
|-------------|---------------------------|-----------|
| id          | 7                         | Уникальный номер (число или текст), не повторять |
| name        | Кожаная куртка            | Название товара |
| brand       | Brunello Cucinelli        | Бренд |
| price       | 120000                    | Цена (только число) |
| currency    | ₽                         | Валюта |
| description | Текст описания...         | Описание |
| category    | clothing                  | Одно из: **clothing** / **shoes** / **bags** / **accessories** |
| images      | /img/p7.jpg               | Путь к фото. Несколько фото: `/img/p7.jpg,/img/p7-2.jpg` (через запятую) |
| specs       | Материал:Кожа\|Цвет:Чёрный | Характеристики: пары «название:значение» через **\|** |
| inStock     | true                      | В наличии: **true** или **false** |
| madeToOrder | false                     | Под заказ: **true** или **false** |
| colors      | Чёрный,Серый              | Цвета через запятую |
| sizes       | 48,50,52                  | Размеры через запятую (или пусто) |
| createdAt   | 2025-03-15T10:00:00Z       | Дата для сортировки новинок (год-месяц-день) |

---

## Шаг 3: Добавить новый товар

1. В конце таблицы добавьте **новую строку**.
2. Заполните все колонки (скопируйте строку существующего товара и измените данные).
3. **id** сделайте уникальным (например 7, 8, 9…).
4. Сохраните файл **как CSV** (разделитель — точка с запятой, если программа спрашивает).

---

## Шаг 4: Обновить каталог на сайте

В папке проекта в терминале выполните:

```bash
npm run products
```

Скрипт сам возьмёт источник данных:
- сначала `data/products.xlsx` (если файл есть),
- иначе `data/products.csv`.

После этого обновите страницу сайта (или перезапустите `npm run dev`) — новые товары появятся в каталоге.

## Картинки (два варианта)

### Вариант 1: Ссылки (URL)

В колонке **images** можно указывать **полную ссылку** на картинку в интернете:

- Одна картинка: `https://example.com/photo.jpg`
- Несколько: `https://example.com/photo1.jpg,https://example.com/photo2.jpg` (через запятую, без пробелов)

Подойдут ссылки с любого сайта, облака (Яндекс.Диск, Google Drive — если файл доступен по прямой ссылке) или фотохостинга.

Важно для фотостоков: ссылка должна быть **прямой на файл изображения** (заканчиваться на `.jpg/.jpeg/.png/.webp`) и разрешать hotlink/внешний доступ.  
Если фотосток блокирует прямую выдачу — картинка не загрузится на сайте.

### Вариант 2: Файлы в проекте

- Кладите фото в папку **`public/img/`** (например `p7.jpg`).
- В колонке **images** укажите: `/img/p7.jpg`.
- Несколько фото: `/img/p7.jpg,/img/p7-2.jpg` (через запятую, без пробелов).

---

## Пример новой строки в таблице

```
7;Кожаная куртка;Brunello Cucinelli;120000;₽;Описание куртки.;clothing;/img/p7.jpg;Материал:Кожа|Цвет:Чёрный;true;false;Чёрный,Коричневый;48,50,52;2025-03-15T10:00:00Z
```

(В CSV одна строка = один товар, колонки разделены точкой с запятой.)

---

## Кратко

1. Редактируете **`data/products.xlsx`** (или `data/products.csv`) в Excel/Google Таблицах.  
2. Запускаете **`npm run products`**.  
3. Каталог на сайте обновлён.
