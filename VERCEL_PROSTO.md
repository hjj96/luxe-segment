# Vercel - Самая простая инструкция

## Почему Vercel?

✅ Создан специально для Next.js  
✅ Всё автоматически  
✅ Бесплатно  
✅ 3 клика - сайт работает  

---

## Что делать (пошагово):

### ШАГ 1: Откройте Vercel

1. Откройте в браузере: **vercel.com**
2. Нажмите **"Sign Up"** или **"Continue with GitHub"**
3. Войдите через ваш GitHub аккаунт

### ШАГ 2: Добавьте проект

1. Нажмите **"Add New Project"** (большая кнопка)
2. Выберите ваш репозиторий: **`hjj96/luxe-segment`**
3. Нажмите **"Import"**

### ШАГ 3: Настройте (опционально)

Vercel автоматически определит Next.js - ничего менять не нужно!

Можно оставить всё как есть.

### ШАГ 4: Деплой!

1. Нажмите **"Deploy"**
2. Подождите 2-3 минуты
3. **ГОТОВО!** Сайт работает!

Вы получите URL типа: `luxe-segment.vercel.app`

---

## Добавить переменные окружения:

После деплоя:

1. В Vercel найдите ваш проект
2. Откройте **Settings** → **Environment Variables**
3. Нажмите **"Add"**
4. Добавьте:
   - **Name**: `JWT_SECRET`
   - **Value**: `ваш-секретный-ключ`
   - **Environment**: Production
5. Нажмите **"Save"**
6. Добавьте `NODE_ENV=production` (аналогично)
7. Перезапустите деплой: **Deployments** → три точки → **Redeploy**

## Если нужен свой домен:

1. В Vercel: **Settings** → **Domains**
2. Добавьте ваш домен
3. Следуйте инструкциям по DNS
4. Готово!
