# Простые сервисы для деплоя (для "тупых") 😊

## Самые простые варианты (рекомендую!)

### 1. Vercel ⭐⭐⭐ (САМЫЙ ПРОСТОЙ!)

**Почему простой:**
- ✅ Автоматический деплой из GitHub
- ✅ Всё настраивается автоматически
- ✅ Не нужно настраивать сервер
- ✅ Бесплатный тариф
- ✅ SSL автоматически
- ✅ CDN включен

**Что делать:**
1. Зайдите на vercel.com
2. Войдите через GitHub
3. Нажмите "Add New Project"
4. Выберите ваш репозиторий `luxe-segment`
5. Нажмите "Deploy"
6. **ГОТОВО!** Сайт работает через 2 минуты

**Цена:** Бесплатно (до 100GB трафика/месяц)

**Подходит для:** Да, идеально для Next.js!

## Что нужно настроить после деплоя

После того как сайт задеплоится на Vercel:

1. **Добавьте переменные окружения:**
   - В Vercel: Settings → Environment Variables
   - Добавьте: `JWT_SECRET=ваш-ключ`
   - Добавьте: `NODE_ENV=production`
   - Если используете Email/SMS - добавьте ключи

2. **Перезапустите деплой:**
   - Deployments → три точки → Redeploy

3. **Готово!** Сайт работает

---

## Если нужен свой домен

После деплоя на Vercel:

1. В Vercel: Settings → Domains
2. Добавьте ваш домен
3. Следуйте инструкциям по настройке DNS
4. Готово!
