# Beget - Простая инструкция

## Почему Beget проще?

✅ Всё через веб-панель - не нужно SSH  
✅ Node.js уже настроен  
✅ Меньше команд  
✅ Русская поддержка  

---

## Что делать:

### 1. Зарегистрируйтесь на beget.com

### 2. Создайте сайт:
- В панели: "Сайты" → "Создать сайт"
- Тип: **"Node.js"**
- Домен: ваш домен

### 3. Загрузите код:
- Откройте "Менеджер файлов"
- Откройте "Терминал" в панели
- Выполните: `git clone https://github.com/hjj96/luxe-segment.git .`

### 4. Установите зависимости:
В терминале: `npm install --production`

### 5. Создайте .env файл:
В менеджере файлов создайте файл `.env` с:
```
JWT_SECRET=ключ
NODE_ENV=production
```

### 6. Соберите проект:
В терминале: `npm run build`

### 7. Запустите через панель:
В разделе "Node.js приложения" создайте приложение и запустите.
