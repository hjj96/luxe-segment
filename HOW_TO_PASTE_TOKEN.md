# Как вставить токен в терминале macOS

## Проблема
При запросе пароля в терминале символы не отображаются, и обычная вставка может не работать.

## Решение

### Способ 1: Вставка через меню (Надежнее)

1. **Скопируйте токен** из GitHub (Ctrl+C или Cmd+C)

2. В терминале, когда появится запрос пароля:
   - **НЕ используйте Cmd+V**
   - Вместо этого: **Правый клик** в поле ввода → **Вставить** (Paste)
   - Или: **Edit** → **Paste** в меню терминала

3. После вставки нажмите **Enter**

### Способ 2: Используйте Shift+Insert (если доступно)

Некоторые терминалы поддерживают:
- **Shift+Insert** для вставки

### Способ 3: Вставьте токен в URL (Избегает ввода)

Если вставка не работает, используйте токен прямо в URL:

```bash
# Удалите текущий remote
git remote remove origin

# Добавьте remote с токеном в URL (замените TOKEN на ваш токен)
git remote add origin https://ghp_ВАШ_ТОКЕН@github.com/hjj96/luxe-segment.git

# Теперь push не будет спрашивать пароль
git push -u origin main
```

⚠️ **Внимание**: Токен будет виден в истории команд. После успешного push можно удалить его из URL:

```bash
# После успешного push, измените обратно на обычный URL
git remote set-url origin https://github.com/hjj96/luxe-segment.git

# И сохраните токен в Keychain для будущего использования
git config --global credential.helper osxkeychain
```

### Способ 4: Используйте SSH вместо HTTPS

Если токен не вставляется, переключитесь на SSH:

```bash
# Удалите HTTPS remote
git remote remove origin

# Добавьте SSH remote
git remote add origin git@github.com:hjj96/luxe-segment.git

# Push через SSH (не будет спрашивать токен)
git push -u origin main
```

Для SSH нужно настроить SSH ключ (см. `GITHUB_AUTH.md`)

---

## Проверка вставки

После вставки токена:
- Символы **НЕ будут видны** (это нормально!)
- Просто нажмите **Enter**
- Если токен правильный, push начнется

---

## Если все еще не работает

### Попробуйте ввести токен вручную:
1. Откройте токен в браузере/текстовом редакторе
2. Поставьте курсор в поле пароля в терминале
3. Вводите токен посимвольно (символы не будут видны - это нормально)
4. Нажмите Enter

### Или используйте способ 3 (токен в URL) - самый надежный!
