# Простая инструкция: Что делать пошагово

## Что у вас есть:
- ✅ Сервер на Reg.ru (название: "Turquoise Promethium")
- ✅ Сервер работает (статус: "Активен")
- ✅ Операционная система: Ubuntu 24.04 LTS
- ✅ Код на GitHub: https://github.com/hjj96/luxe-segment

## Что нужно сделать:
Загрузить код с GitHub на сервер и запустить сайт.

---

## ШАГ 1: Найти IP адрес сервера

### Что делать:
1. В панели Reg.ru найдите вкладку **"Сеть"** (рядом с "Информация")
2. Нажмите на вкладку **"Сеть"**
3. Найдите там **"IP адрес"** или **"Внешний IP"**
4. **Скопируйте этот IP адрес** (например: 149.88.109.83)
5. Запишите его - он понадобится

---

## ШАГ 2: Найти пароль для входа на сервер

### Что делать:
1. В панели Reg.ru найдите кнопку **"..."** (три точки) рядом с кнопками "Остановить сервер"
2. Нажмите на **"..."** → **"Пароли"** или **"Доступ"**
3. Найдите пароль для пользователя **"root"**
4. Если пароля нет - создайте новый:
   - Нажмите **"Сбросить пароль"** или **"Создать пароль"**
   - Запишите пароль - он понадобится

**ИЛИ:**

1. Найдите в панели раздел **"Консоль"** или **"VNC"** или **"Веб-консоль"**
2. Нажмите на него - откроется окно с терминалом сервера
3. Это проще, чем SSH!

---

## ШАГ 3: Подключиться к серверу

### Вариант A: Через веб-консоль (ПРОЩЕ!)

1. В панели Reg.ru найдите **"Консоль"** или **"VNC"** или **"Веб-консоль"**
2. Нажмите на эту кнопку
3. Откроется окно с черным экраном (терминал)
4. **Готово!** Вы подключены к серверу

### Вариант B: Через SSH (если веб-консоли нет)

1. На вашем Mac откройте программу **"Терминал"** (Terminal)
2. Введите команду (замените IP на ваш):
   ```
   ssh root@149.88.109.83
   ```
3. Нажмите Enter
4. Если спросит пароль - введите пароль от root (который вы нашли в шаге 2)
5. Если спросит "Are you sure?" - введите `yes` и нажмите Enter

## ШАГ 4: Выполнить команды на сервере

**ВАЖНО:** Теперь вы на сервере! Вы увидите что-то вроде:
```
root@server:~#
```

### Что делать:
Скопируйте и вставьте команды по очереди (каждую команду - отдельно, нажимайте Enter после каждой):

### Команда 1: Обновить систему
```bash
apt update && apt upgrade -y
```
Нажмите Enter, подождите пока выполнится (может занять 2-5 минут)

### Команда 2: Установить Node.js
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
```
Нажмите Enter, подождите

### Команда 3: Установить Node.js (продолжение)
```bash
apt install -y nodejs
```
Нажмите Enter, подождите

### Команда 4: Установить PM2
```bash
npm install -g pm2
```
Нажмите Enter, подождите

### Команда 5: Установить Nginx и Git
```bash
apt install -y nginx git
```
Нажмите Enter, подождите

## ШАГ 5: Загрузить код с GitHub

### Команда 1: Создать папку для проекта
```bash
mkdir -p /var/www/luxe-segment
```
Нажмите Enter

### Команда 2: Перейти в папку
```bash
cd /var/www/luxe-segment
```
Нажмите Enter

### Команда 3: Скачать код с GitHub
```bash
git clone https://github.com/hjj96/luxe-segment.git .
```
Нажмите Enter, подождите (скачает код)

## ШАГ 6: Настроить проект

### Команда 1: Установить зависимости
```bash
npm install --production
```
Нажмите Enter, подождите (может занять 2-3 минуты)

### Команда 2: Создать файл с настройками
```bash
nano .env
```
Нажмите Enter - откроется редактор

### Что ввести в редакторе:
Скопируйте и вставьте:
```
JWT_SECRET=ваш-секретный-ключ-здесь
NODE_ENV=production
```

**Как вставить:**
- Правый клик мыши → Вставить
- Или: Ctrl+Shift+V

**Как сохранить:**
- Нажмите: `Ctrl+O` (буква O, не ноль)
- Нажмите: `Enter`
- Нажмите: `Ctrl+X` (закрыть редактор)

### Команда 3: Собрать проект
```bash
npm run build
```
Нажмите Enter, подождите (может занять 1-2 минуты)

---

## ШАГ 7: Запустить сайт

### Команда 1: Запустить через PM2
```bash
pm2 start ecosystem.config.js
```
Нажмите Enter

### Команда 2: Сохранить настройки PM2
```bash
pm2 save
```
Нажмите Enter

### Команда 3: Настроить автозапуск
```bash
pm2 startup
```
Нажмите Enter

**ВАЖНО:** PM2 выведет команду типа:
```
sudo env PATH=... pm2 startup systemd -u root --hp /root
```

**Скопируйте эту команду полностью и выполните ее!**

## ШАГ 8: Настроить Nginx (чтобы сайт был доступен)

### Команда 1: Создать файл конфигурации
```bash
nano /etc/nginx/sites-available/luxe-segment
```
Нажмите Enter

### Что ввести:
Скопируйте и вставьте (замените `ваш-домен.ru` на ваш домен):
```
server {
    listen 80;
    server_name ваш-домен.ru www.ваш-домен.ru;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }
}
```

**Как сохранить:**
- `Ctrl+O` → `Enter` → `Ctrl+X`

### Команда 2: Включить конфигурацию
```bash
ln -s /etc/nginx/sites-available/luxe-segment /etc/nginx/sites-enabled/
```
Нажмите Enter

### Команда 3: Проверить конфигурацию
```bash
nginx -t
```
Нажмите Enter - должно показать "syntax is ok"

### Команда 4: Перезапустить Nginx
```bash
systemctl restart nginx
```
Нажмите Enter

## ШАГ 9: Установить SSL (чтобы был HTTPS)

### Команда 1: Установить Certbot
```bash
apt install -y certbot python3-certbot-nginx
```
Нажмите Enter, подождите

### Команда 2: Получить SSL сертификат
```bash
certbot --nginx -d ваш-домен.ru -d www.ваш-домен.ru
```
Нажмите Enter

**Следуйте инструкциям:**
- Введите email
- Согласитесь с условиями (введите `A` или `Y`)
- Выберите редирект HTTP → HTTPS (введите `2` или `Y`)

## ШАГ 10: Настроить DNS в Reg.ru

### Что делать:
1. В панели Reg.ru найдите раздел **"Домены"**
2. Выберите ваш домен
3. Найдите **"DNS-серверы и зона"** или **"Управление DNS"**
4. Добавьте **A-запись**:
   - **Тип**: A
   - **Имя**: @ (или оставьте пустым)
   - **Значение**: IP адрес вашего сервера (который вы нашли в шаге 1)
   - **TTL**: 3600
5. Добавьте еще одну **A-запись** для www:
   - **Тип**: A
   - **Имя**: www
   - **Значение**: тот же IP адрес
   - **TTL**: 3600
6. Сохраните изменения

## Если что-то не работает

### Не могу подключиться к серверу:
- Используйте веб-консоль в панели Reg.ru (проще!)
- Или проверьте пароль root в панели

### Команда не работает:
- Убедитесь, что вы на сервере (видите `root@server:~#`)
- Не выполняйте команды на Mac - только на сервере!

### Сайт не открывается:
- Подождите 10-15 минут (DNS обновляется)
- Проверьте, что сайт запущен: `pm2 status`
- Проверьте логи: `pm2 logs luxe-segment`
