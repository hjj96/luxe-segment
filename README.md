# Luxe Segment

Минималистичный каталог премиального сегмента. Консультации и заказы — через Telegram.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Сборка

```bash
npm run build
npm start
```

## Структура

- **Главная** — новинки, видео-ряд, категории, блоки Доставка / Оплата / Гарантия
- **Каталог** — фильтры (категория, бренд, цена, цвет, размер, наличие), на мобильном — нижняя панель (bottom sheet)
- **Карточка товара** — галерея, описание, избранное, корзина, CTA в Telegram
- **Избранное** — localStorage, без авторизации
- **Корзина** — оформление заявки (имя, телефон, адрес, комментарий, способ доставки)
- **Аккаунт** — вход по телефону/email с OTP (демо)
- **Доставка, Оплата, Гарантия** — информационные страницы

Ссылки на Telegram (канал и чат) вынесены в шапку и на страницу благодарности после заказа.
