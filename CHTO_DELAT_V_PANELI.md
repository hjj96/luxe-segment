# Что делать в панели управления сервером

## Что вы видите в панели:

### 1. **Реквизиты доступа** (слева вверху)
- **Подключение**: `root@45.67.59.86` - это данные для входа
- **Внешний IP**: `45.67.59.86` - это адрес вашего сервера

### 2. **Файловый менеджер** (справа вверху)
- Нажмите на этот блок
- Откроется файловый менеджер - можно загружать файлы

### 3. **Терминал VNC** (слева внизу)
- **НАЖМИТЕ СЮДА!** ⭐
- Откроется терминал сервера
- Здесь можно выполнять команды

### 4. **Мониторинг** (справа внизу)
- Графики использования сервера
- Показывает нагрузку CPU и память

---

## Что делать СЕЙЧАС:

### ШАГ 1: Откройте терминал

1. Найдите блок **"Терминал VNC"** (слева внизу)
2. **Нажмите на него**
3. Откроется чёрное окно с терминалом

### ШАГ 2: Выполните команды в терминале

В открывшемся терминале выполните команды по очереди:

#### Проверьте систему:
```bash
cat /etc/os-release
```

Это покажет какая система установлена.

#### Если Ubuntu/Debian:
```bash
apt update && apt upgrade -y
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install -y nodejs npm
npm install -g pm2
apt install -y nginx git
```

#### Если CentOS/RHEL:
```bash
yum update -y
curl -fsSL https://rpm.nodesource.com/setup_20.x | bash -
yum install -y nodejs npm
npm install -g pm2
yum install -y nginx git
```

### ШАГ 3: Загрузите код с GitHub

```bash
mkdir -p /var/www/luxe-segment
cd /var/www/luxe-segment
git clone https://github.com/hjj96/luxe-segment.git .
```

### ШАГ 4: Настройте проект

```bash
npm install --production
nano .env
```

В редакторе вставьте:
```
JWT_SECRET=ваш-секретный-ключ
NODE_ENV=production
```

Сохраните: `Ctrl+O`, `Enter`, `Ctrl+X`

```bash
npm run build
```

### ШАГ 5: Запустите сайт

```bash
pm2 start ecosystem.config.js
pm2 save
pm2 startup
```

Выполните команду, которую выведет PM2.

2. Загрузите файлы проекта через веб-интерфейс
3. Но для установки Node.js всё равно нужен терминал

## Что делать прямо сейчас:

1. **Нажмите на "Терминал VNC"**
2. **Выполните**: `cat /etc/os-release`
3. **Пришлите результат** - скажу точные команды

Или попробуйте:

```bash
apt update
```

Если работает - используйте команды с `apt`.  
Если не работает - используйте команды с `yum`.

---

## Если всё равно сложно:

Можно использовать **Beget** - там действительно проще:
- Всё через веб-панель
- Node.js уже настроен
- Меньше команд

Но если хотите использовать этот сервер - начните с "Терминал VNC" и выполните команды выше.
