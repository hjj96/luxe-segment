# Деплой на Beget - Простая инструкция

## Почему Beget проще?

✅ **Встроенная поддержка Node.js** - не нужно настраивать вручную  
✅ **Простая панель управления** - всё через веб-интерфейс  
✅ **Автоматическая настройка** - меньше команд в терминале  
✅ **Русская поддержка** - легко получить помощь  

---

## Шаг 1: Создать аккаунт на Beget

1. Откройте https://beget.com
2. Зарегистрируйтесь или войдите
3. Выберите тариф с поддержкой Node.js (обычно от 200₽/месяц)

---

## Шаг 2: Создать сайт в панели

1. Войдите в панель управления Beget
2. Найдите раздел **"Сайты"** или **"Домены"**
3. Нажмите **"Создать сайт"** или **"Добавить сайт"**
4. Заполните:
   - **Домен**: ваш домен (или поддомен для теста)
   - **Тип сайта**: выберите **"Node.js"** или **"Приложение Node.js"**
5. Нажмите **"Создать"**

---

## Шаг 3: Настроить Node.js приложение

### Вариант A: Через панель управления (проще)

1. В панели Beget найдите ваш сайт
2. Откройте настройки сайта
3. Найдите раздел **"Node.js"** или **"Приложения"**
4. Настройте:
   - **Версия Node.js**: выберите 20.x или последнюю
   - **Стартовый файл**: `server.js` или `package.json`
   - **Порт**: обычно автоматически (3000)
   - **Рабочая директория**: `/home/uXXXXXX/ваш-домен.ru/public_html`
5. Сохраните настройки

### Вариант B: Через SSH (если нужно больше контроля)

1. В панели Beget найдите **"SSH доступ"**
2. Скопируйте данные для подключения:
   - Хост: `uXXXXXX.beget.tech` (где X - ваш ID)
   - Пользователь: `uXXXXXX`
   - Пароль: (установите в панели)

---

## Шаг 4: Загрузить код на сервер

### Способ 1: Через Git (рекомендуется)

1. В панели Beget найдите **"Менеджер файлов"** или **"FTP"**
2. Откройте папку вашего сайта (обычно `public_html`)
3. Откройте **"Терминал"** или **"SSH"** в панели
4. Выполните:

```bash
cd ~/ваш-домен.ru/public_html
git clone https://github.com/hjj96/luxe-segment.git .
```

### Способ 2: Через FTP

1. В панели Beget найдите **"FTP доступ"**
2. Используйте FTP клиент (FileZilla) или загрузку через панель
3. Загрузите все файлы проекта в `public_html`

---

## Шаг 5: Установить зависимости

В терминале Beget (или через SSH):

```bash
cd ~/ваш-домен.ru/public_html
npm install --production
```

---

## Шаг 6: Создать файл .env

В панели Beget:

1. Откройте **"Менеджер файлов"**
2. Перейдите в папку сайта
3. Создайте файл `.env`
4. Добавьте:

```
JWT_SECRET=ваш-секретный-ключ
NODE_ENV=production
```

Сохраните файл.

---

## Шаг 7: Собрать проект

В терминале:

```bash
cd ~/ваш-домен.ru/public_html
npm run build
```

---

## Шаг 8: Настроить запуск приложения

### Если Beget использует PM2:

В панели Beget найдите раздел **"Node.js приложения"** или **"PM2"**:

1. Создайте новое приложение
2. Укажите:
   - **Название**: `luxe-segment`
   - **Скрипт**: `npm start` или `node server.js`
   - **Директория**: `~/ваш-домен.ru/public_html`
3. Сохраните и запустите

### Если нужно настроить вручную:

В терминале:

```bash
cd ~/ваш-домен.ru/public_html
pm2 start ecosystem.config.js
pm2 save
```

---

## Шаг 9: Настроить домен

1. В панели Beget найдите **"Домены"**
2. Выберите ваш домен
3. Настройте DNS (если домен не на Beget):
   - Добавьте A-запись на IP сервера Beget
   - Или используйте NS-серверы Beget

---

## Шаг 10: Проверить работу

Откройте в браузере: `https://ваш-домен.ru`

---

## Преимущества Beget

✅ **Проще настройка** - всё через панель  
✅ **Автоматический SSL** - настраивается автоматически  
✅ **Меньше команд** - больше через веб-интерфейс  
✅ **Русская поддержка** - легко получить помощь  
✅ **Готовые шаблоны** - можно выбрать готовую конфигурацию  

---

## Если что-то не работает

1. **Проверьте логи** в панели Beget → "Логи" или "Node.js логи"
2. **Обратитесь в поддержку** - Beget имеет русскую поддержку
3. **Проверьте настройки Node.js** в панели

---

## Сравнение: Beget vs Reg.ru VPS

| Параметр | Beget | Reg.ru VPS |
|----------|-------|------------|
| Сложность | ⭐ Проще | ⭐⭐⭐ Сложнее |
| Настройка | Через панель | Вручную через SSH |
| Node.js | Встроенная поддержка | Нужно устанавливать |
| SSL | Автоматически | Нужно настраивать |
| Поддержка | Русская, быстрая | Зависит от тарифа |
| Цена | От 200₽/мес | От 300₽/мес |

## Рекомендация

**Для начала используйте Beget** - это действительно проще:
- Меньше команд в терминале
- Всё через веб-панель
- Автоматическая настройка Node.js
- Легче получить помощь

После того как сайт заработает, можно перейти на VPS для большего контроля.

---

## Быстрый старт на Beget

1. Зарегистрируйтесь на beget.com
2. Создайте сайт с типом "Node.js"
3. Клонируйте проект через Git
4. Установите зависимости: `npm install --production`
5. Создайте `.env` файл
6. Соберите проект: `npm run build`
7. Запустите через панель Beget
8. Готово!

**Намного проще, чем на VPS!** 🎉
