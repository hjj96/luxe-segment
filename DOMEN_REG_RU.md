# Как подключить домен из Reg.ru к Vercel

## Пошаговая инструкция

### ШАГ 1: Получить данные от Vercel

После деплоя на Vercel:

1. В Vercel откройте ваш проект
2. Перейдите в **Settings** → **Domains**
3. Нажмите **"Add"** или **"Add Domain"**
4. Введите ваш домен (например: `luxesegment.ru`)
5. Vercel покажет что нужно настроить DNS

Vercel покажет вам:
- **A-запись** или **CNAME-запись**
- **Значение** (IP адрес или домен Vercel)

**Пример:**
- Тип: CNAME
- Имя: @ (или пусто)
- Значение: `cname.vercel-dns.com`

Или:
- Тип: A
- Имя: @
- Значение: `76.76.21.21` (IP адрес Vercel)

---

### ШАГ 2: Настроить DNS в Reg.ru

1. **Войдите в панель Reg.ru**
   - Откройте reg.ru
   - Войдите в аккаунт

2. **Перейдите в управление доменом**
   - Найдите раздел **"Домены"**
   - Выберите ваш домен
   - Нажмите **"DNS-серверы и зона"** или **"Управление DNS"**

3. **Добавьте записи**

#### Вариант A: Если Vercel дал CNAME

Добавьте CNAME запись:
- **Тип**: CNAME
- **Имя**: @ (или оставьте пустым для корня домена)
- **Значение**: `cname.vercel-dns.com` (или что дал Vercel)
- **TTL**: 3600 (или оставьте по умолчанию)
- Нажмите **"Добавить"**

**ВАЖНО:** Если CNAME для корня (@) не поддерживается, используйте A-запись (см. ниже)

#### Вариант B: Если Vercel дал A-запись

Добавьте A-запись:
- **Тип**: A
- **Имя**: @ (или оставьте пустым)
- **Значение**: IP адрес от Vercel (например: `76.76.21.21`)
- **TTL**: 3600
- Нажмите **"Добавить"**

#### Для поддомена www:

Добавьте еще одну запись:
- **Тип**: CNAME
- **Имя**: www
- **Значение**: `cname.vercel-dns.com` (или что дал Vercel)
- **TTL**: 3600
- Нажмите **"Добавить"**

---

### ШАГ 3: Подождать обновления DNS

После добавления записей:
- Подождите **5-30 минут** (обычно 10-15 минут)
- DNS записи должны обновиться

---

### ШАГ 4: Проверить в Vercel

1. Вернитесь в Vercel
2. В разделе **Settings** → **Domains**
3. Vercel проверит подключение
4. Когда увидите зелёную галочку ✅ - домен подключен!

---

## Подробная инструкция для Reg.ru

### Где найти настройки DNS в Reg.ru:

1. **Войдите в панель Reg.ru**
2. В верхнем меню найдите **"Домены"**
3. Найдите ваш домен в списке
4. Нажмите на домен
5. В меню домена найдите:
   - **"DNS-серверы и зона"**
   - Или **"Управление DNS"**
   - Или **"Зона DNS"**
6. Откроется список DNS записей

### Как добавить запись:

1. Нажмите **"Добавить запись"** или **"+"**
2. Выберите тип записи (A или CNAME)
3. Заполните:
   - **Имя**: @ (для корня) или www (для поддомена)
   - **Значение**: то, что дал Vercel
   - **TTL**: 3600 (или по умолчанию)
4. Нажмите **"Сохранить"** или **"Добавить"**

## Примеры настроек

### Пример 1: CNAME для корня (если поддерживается)

В Reg.ru:
- Тип: **CNAME**
- Имя: **@** (или пусто)
- Значение: **cname.vercel-dns.com**
- TTL: **3600**

### Пример 2: A-запись для корня

В Reg.ru:
- Тип: **A**
- Имя: **@** (или пусто)
- Значение: **76.76.21.21** (IP от Vercel)
- TTL: **3600**

### Пример 3: CNAME для www

В Reg.ru:
- Тип: **CNAME**
- Имя: **www**
- Значение: **cname.vercel-dns.com**
- TTL: **3600**

---

## Если Reg.ru не поддерживает CNAME для корня (@)

Некоторые регистраторы не поддерживают CNAME для корня домена.

**Решение:**
1. Используйте **A-запись** вместо CNAME для корня
2. Vercel даст вам IP адрес для A-записи
3. Добавьте A-запись в Reg.ru

---

## Проверка DNS

После настройки можно проверить:

1. **Через Vercel:**
   - Settings → Domains
   - Vercel покажет статус подключения

2. **Через онлайн-сервисы:**
   - Откройте: https://dnschecker.org
   - Введите ваш домен
   - Проверьте записи

3. **Через терминал (на Mac):**
   ```bash
   dig ваш-домен.ru
   ```
   Или:
   ```bash
   nslookup ваш-домен.ru
   ```

## Частые проблемы

### Проблема: "DNS не обновился"

**Решение:**
- Подождите до 24 часов (обычно 10-15 минут)
- Очистите кэш DNS на вашем компьютере
- Проверьте правильность записей в Reg.ru

### Проблема: "CNAME для корня не поддерживается"

**Решение:**
- Используйте A-запись вместо CNAME
- Vercel даст IP адрес для A-записи

### Проблема: "Домен не подключается"

**Решение:**
- Проверьте правильность значений в DNS записях
- Убедитесь, что записи сохранены в Reg.ru
- Подождите обновления DNS (до 24 часов)

---

## Быстрая инструкция

1. **В Vercel:** Settings → Domains → Add → Введите домен
2. **Скопируйте** данные DNS от Vercel
3. **В Reg.ru:** Домены → Ваш домен → DNS-серверы и зона
4. **Добавьте** записи (A или CNAME) с данными от Vercel
5. **Подождите** 10-15 минут
6. **Проверьте** в Vercel - должна быть зелёная галочка ✅

---

## Готово! 🎉

После настройки DNS ваш сайт будет доступен по вашему домену:
- `https://ваш-домен.ru`
- `https://www.ваш-домен.ru`

Vercel автоматически выдаст SSL сертификат!
