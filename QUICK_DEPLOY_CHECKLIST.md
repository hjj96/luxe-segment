# Чеклист деплоя на Reg.ru

## ✅ Что делать ГДЕ

### На вашем Mac (локально):

- [ ] Подключиться к серверу: `ssh root@149.88.109.83`
- [ ] Ввести пароль от сервера (если спросит)

### На сервере Reg.ru (после подключения):

- [ ] Обновить систему: `apt update && apt upgrade -y`
- [ ] Установить Node.js: `curl -fsSL https://deb.nodesource.com/setup_20.x | bash - && apt install -y nodejs`
- [ ] Установить PM2: `npm install -g pm2`
- [ ] Установить Nginx: `apt install -y nginx git`
- [ ] Клонировать проект: `git clone https://github.com/hjj96/luxe-segment.git /var/www/luxe-segment`
- [ ] Перейти в проект: `cd /var/www/luxe-segment`
- [ ] Установить зависимости: `npm install --production`
- [ ] Создать .env файл: `nano .env`
- [ ] Собрать проект: `npm run build`
- [ ] Запустить через PM2: `pm2 start ecosystem.config.js`
- [ ] Сохранить PM2: `pm2 save`
- [ ] Настроить автозапуск: `pm2 startup`
- [ ] Настроить Nginx (скопировать из nginx.conf.example)
- [ ] Установить SSL: `certbot --nginx -d ваш-домен.ru`
- [ ] Настроить DNS в панели Reg.ru

## 🔍 Как понять, что вы на сервере

После подключения через SSH вы увидите:
```
root@server:~#
```

Или:
```
root@vps123456:~#
```

**НЕ должно быть:**
```
a@MacBook-Air-Andrey ЛСсайт %
```

Если видите "MacBook-Air-Andrey" - вы на Mac, нужно подключиться к серверу!

## ⚠️ Частые ошибки

### Ошибка: "command not found: apt"
**Причина:** Вы на Mac, а не на сервере  
**Решение:** Подключитесь к серверу через SSH

### Ошибка: "Permission denied (publickey)"
**Причина:** SSH не может подключиться  
**Решение:** 
- Используйте пароль (установите в панели Reg.ru)
- Или используйте веб-консоль в панели Reg.ru

### Ошибка: "EACCES: permission denied" при npm install -g
**Причина:** Вы на Mac, пытаетесь установить глобально  
**Решение:** Это нормально для Mac, но PM2 нужно устанавливать НА СЕРВЕРЕ

## 🆘 Если не получается подключиться

1. Проверьте IP адрес в панели Reg.ru
2. Проверьте пароль root в панели Reg.ru
3. Используйте веб-консоль в панели Reg.ru
4. Обратитесь в поддержку Reg.ru
